While right button is pressed ¡ú use mouse movement in x direction to pan;
and
While middle button is pressed ¡ú use mouse movement in y direction to tilt.

The user can zoom in and out of the scene - while left button is pressed ¡ú use
mouse movement to move into/out of the scene.

The world orientation is changed by using keyboard input i.e. left arrow ¡ú Rx,
right arrow ¡ú R-x, up arrow ¡ú Ry, down arrow ¡ú R-y.

A ¡ú move left, D ¡ú move right, W ¡ú move up, S ¡ú move down.

Z ¡ú rotate left 5 degrees about Y axis, X ¡ú rotate right 5 degrees about Y axis.
